# Registration Form

This is a simple registration form designed using HTML and styled with CSS. The form collects basic information from users, such as their first name, last name, email, password, account type, profile picture, age, referrer, and bio. It also includes a checkbox for accepting terms and conditions.
## Features

- Responsive design: The form is designed to be responsive and adapt to different screen sizes.
- Required fields: Certain fields are marked as required, ensuring that users provide essential information.
- Input validation: The password field has a pattern attribute to enforce a specific pattern for the password.
